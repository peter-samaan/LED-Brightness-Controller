# LED-Brightness-Controller

## Overview
Pairs with SCOMP architecture to allow manual adjustment of LED brightness on an FPGA using embedded VHDL and assembly code, tested on DE-10 Standard Board

## Features
- Allows manual LED control: both controlling brightness levels and targetting specific LEDs
- Gamma-correction LUT enhances "linear feel" in progressive brightness levels
- demo assembly file that creates temporary light wave effect and uses external switches to visualize bit manipulation for dynamic LED control

## Tools
- VHDL
- Assembly (somewhat scuffed)
- DE-10 Standard FPGA Board
- Intel Altera Quartus Software
